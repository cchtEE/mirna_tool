miR-Gyn Explorer Tutorial

1.1. Differential miRNA Analysis

In this section, User can select the cancer type of choice and the stage wise-comparion. For each cancer compariosn a table of miRNA regulation statistics will be loaded on the page. Volcanoplot and expression of a single miRNA in the sample groups can be visualized. There are two options for stage wise comparion, (1) Normal vs Stage I and (2) Between all stages includes, stage I-IV. in case of first option (i.e. Normal vs Stage I)- User can load the differentially regulated miRNA statistics, Volcano plot and miRNA specific Box plot.

1.2. Differential expression Between stages of Cancer

In addition to above mentioned results, in this section user can explore the survial analysis for each miRNA between clinical stages of cancer. Cox-proportional hazard model and log rank test are perfomed for each miRNA expression and can be loaded from the drop-down menu. Kaplan-Meier Plot can be generated for the miRNA.

1.3. Validated targets of differentially regulated miRNAs

In the second tab of the first page "miRNA-Target", validated miRNA targets from MiRTarBase, miRecords and TarBase databases can be reterived. The usre can select the targets accroding to the database. Three database options are available to select from the menu. This section retrieve the targets of differentially regulated miRNAs for each cancer type and between the normal and the stage wise comparions.

The action button, "Show miRNA target interactions" retrives the validated interactions of differentially regulated miRNAs and mRNAs for each cancer. similarly for differentially regulated miRNA and lncRNA intercations can be loaded in case of miRNA-lncRNA.

The action button, "miRNA target function annotations and Protein expression" part shows the Functional enrichment analysis, where upto Top 10 GO terms and upto top 20 KEGG pathways for each comparion. The third option from the drop-down menu retrives the protein expression in cancer tissue information from The Human Protein Atlas database. protein expression in cancer tissue of miRNA target genes can be viewed by following the link from the table. The table shows the prognostic effect of a miRNA targets in given cancer tissue.

1.4. miRNA Network

The third tab on first page is about miRNA-mRNA-lncRNA network and hub network modules. This section generates the network form differentially regulated miRNA, mRNA and lncRNAs. In this section, User can specify the correlation coeffiecient between 0.5-1. Only highly correlated miRNA-gene paires are used to generate the network. The final network generated is submitted to fastgreedy method to identify hub network modules and each module has a different color. Users can click on the network nodes to visualize the correlation coeffiecint between the selected nodes. Network assortivity coeffiecint foe each network is also calculated. The list of network froming genes with miRNas can be downloaed to submit for any futher specific analysis, such as, the list can be uploaded to stringDB.
